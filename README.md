### Что умеет MetaFlow?
1. Создание учетных записей для организации и сотрудников
2. Управление распорядком дня сотрудника и распределение ролей
3. Просматривать, создавать, редактировать и удалять задачи
4. Уведомления для сотрудников
5. Интеграция с почтой

### Какие технологии мы используем?
| Категория  | Стэк |
|-----------|-------|
| Frontend  | Blazor Web App (.NET 9), MudBlazor, TailwindCSS + DaisyUI, Fluxor |
| Backend   | ASP.NET Core 9 (Minimal APIs), Carter, CQRS + MediatR, Mapster |
| Database  | PostgreSQL 16 (Supabase), EF Core 9, Dapper |
| Auth      | Duende IdentityServer / Keycloak (OIDC, OAuth2) |
| Email     | Postmark, Resend, FluentEmail |
